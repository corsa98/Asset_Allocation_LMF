O projeto busca automatizar a busca por ativos da bolsa brasileira e fazer análises de uma carteira de investimentos, como a simulação de monte carlo, a correlação dos ativos e a sua comparação com alguns indicadores como a ibovespa.

Planejo melhorar esse código com o tempo, mas se você tiver qualquer forma de me ajudar com isso, será bem-vindo, a maioria do código foi inspirada no projeto de Thiago Teixeira: https://github.com/streamlit/demo-stockpeers/blob/main/streamlit_app.py
